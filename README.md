## Prerequisite

[Node.js](https://nodejs.org/)

## Installation

```javascript
npm install
```

## Usage

Run as development server:

```javascript
npm start
```

Run as Electron appliaction on development environment:

```javascript
npm run electron-dev
```

Build as web application:

```javascript
npm build
```

Build as Electron application:

```javascript
npm run electron-pack
```
